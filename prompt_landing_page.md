# Prompt para Generar una Landing Page Profesional

## Objetivo

Crea una landing page profesional, moderna y responsive para un negocio del sector:

```text
[SECTOR DEL NEGOCIO]
```

La landing debe estar diseñada para generar contactos, transmitir confianza y conseguir conversiones mediante un CTA claro y visible.

---

## Informacion del Negocio

Completa los siguientes datos antes de generar la landing page:

```text
Nombre del negocio: [NOMBRE DEL NEGOCIO]
Direccion: [DIRECCION]
Telefono: [TELEFONO]
Correo electronico: [CORREO]
Ciudad o zona de servicio: [CIUDAD O ZONA]
Horario: [HORARIO]
Servicios principales: [SERVICIOS]
Publico objetivo: [CLIENTE IDEAL]
```

---

## Prompt Principal

```text
Crea una landing page profesional, moderna y responsive para un negocio del sector: [SECTOR DEL NEGOCIO].

Informacion del negocio:
- Nombre: [NOMBRE DEL NEGOCIO]
- Direccion: [DIRECCION]
- Telefono: [TELEFONO]
- Correo electronico: [CORREO]
- Ciudad / zona de servicio: [CIUDAD O ZONA]
- Horario: [HORARIO]
- Servicios principales: [SERVICIOS]
- Publico objetivo: [CLIENTE IDEAL]

Objetivo de la landing:
Generar contactos y conversiones. La pagina debe tener un CTA claro y visible, por ejemplo: "Llamar ahora", "Pedir presupuesto", "Reservar cita" o "Contactar por WhatsApp".

Estilo visual:
Define un estilo acorde al sector del negocio. El diseno debe transmitir confianza, profesionalidad y coherencia con el tipo de empresa.

Adapta automaticamente:
- Paleta de colores segun el sector.
- Tipografia adecuada al tono del negocio.
- Estilo visual: elegante, cercano, tecnico, premium, artesanal, sanitario, corporativo, etc.
- Iconografia relacionada con el sector.
- Imagen hero relacionada directamente con el sector del negocio.

Estructura obligatoria:
1. Header con logo/nombre del negocio, navegacion simple y CTA.
2. Hero principal con:
   - Titular potente orientado al beneficio.
   - Subtitulo explicando que ofrece el negocio.
   - CTA principal visible.
   - Imagen hero relacionada con el sector.
3. Seccion de beneficios o ventajas competitivas.
4. Seccion de servicios principales.
5. Seccion "Por que elegirnos".
6. Testimonios o prueba social. Si no hay datos reales, usar ejemplos realistas.
7. Seccion de contacto con telefono, correo, direccion y horario.
8. Footer con datos del negocio.

Requisitos de diseno:
- Diseno responsive para movil, tablet y escritorio.
- Estetica limpia, moderna y comercial.
- Buen contraste y legibilidad.
- CTA repetido en varias zonas estrategicas.
- Evitar textos genericos; usar copywriting especifico del sector.
- La imagen del hero debe ser realista y coherente con el negocio.
- No usar librerias pesadas ni animaciones complejas.

Requisitos tecnicos:
Genera el resultado como una landing page completa en HTML, CSS y JavaScript simple, optimizada para ejecutarse en un entorno local con una RTX 6030 de 8GB de VRAM. Mantener el codigo ligero, sin dependencias innecesarias y sin frameworks pesados.

Incluye tambien un prompt separado para generar la imagen del hero con IA, optimizado para el sector indicado.

Devuelve:
1. El codigo completo de la landing page.
2. El prompt para la imagen hero.
3. Una breve explicacion del estilo elegido y por que encaja con el sector.
```

---

## Prompt para la Imagen Hero

Usa este prompt para generar la imagen principal de la landing page:

```text
Imagen hero realista para una landing page de [SECTOR DEL NEGOCIO].

Debe mostrar [DESCRIPCION DEL NEGOCIO O SERVICIO PRINCIPAL] en un entorno profesional, limpio y atractivo.

Estilo visual coherente con una marca [ELEGANTE / CERCANA / PREMIUM / TECNICA / ARTESANAL / SANITARIA].

Iluminacion natural, composicion moderna, alta calidad, enfoque comercial, sin texto, sin logos, formato horizontal 16:9.
```

---

## Recomendaciones para una RTX 6030 con 8 GB de VRAM

Para generar o trabajar con este contenido en un equipo con 8 GB de VRAM, conviene mantener el proyecto ligero:

- Usar HTML, CSS y JavaScript simple.
- Evitar frameworks pesados si no son necesarios.
- Generar imagenes en resoluciones moderadas, por ejemplo 1024x576 o 1280x720.
- Optimizar las imagenes antes de usarlas en la landing.
- Evitar animaciones complejas, videos pesados o efectos 3D.
- Priorizar carga rapida, buena legibilidad y estructura clara.

---

## Checklist de Calidad

Antes de dar por terminada la landing, comprobar:

- El CTA principal es claro y visible.
- La imagen hero esta relacionada con el sector.
- El estilo visual encaja con el tipo de negocio.
- Los datos de contacto son faciles de encontrar.
- La pagina funciona correctamente en movil y escritorio.
- El texto esta orientado a beneficios, no solo a descripcion de servicios.
- El codigo es ligero y facil de mantener.
